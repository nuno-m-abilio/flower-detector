# Referência de Características Botânicas — Sistema Especialista de Flores
## Angiospermas (plantas com flores)

> Este arquivo descreve a hierarquia de características usadas para identificar e catalogar flores. Cada característica pode ser usada como filtro na consulta.

---

# 1. Flores

## 1.1 Cor das Pétalas

A cor é determinada principalmente por pigmentos (flavonoides, carotenoides, clorofila) e pelo pH celular. Muitas flores apresentam variações graduais ou padrões multicoloridos — nesses casos, registrar a cor dominante e as secundárias separadamente.

### Amarelo

| Variação | Descrição |
|---|---|
| Amarelo-palha | Tom muito claro, quase branco-amarelado. *Ex: primrose pálida.* |
| Amarelo-creme | Suave, levemente quente. *Ex: rosa creme, magnólia.* |
| Amarelo-limão | Amarelo claro com toque esverdeado. *Ex: algumas orquídeas.* |
| Amarelo-dourado | Amarelo saturado, vibrante. *Ex: girassol, margarida-amarela.* |
| Amarelo-âmbar | Amarelo com toque alaranjado profundo. *Ex: calêndula.* |
| Amarelo-mostarda | Amarelo escuro, terroso. *Ex: algumas variedades de crisântemo.* |

### Laranja

| Variação | Descrição |
|---|---|
| Laranja-pêssego | Tom suave, rosado. *Ex: rosa salmão, gérbera pêssego.* |
| Laranja-claro | Laranja pastel. *Ex: algumas tulipas.* |
| Laranja-vivo | Laranja saturado e brilhante. *Ex: gérbera laranja, calêndula.* |
| Laranja-cobre | Laranja escuro com brilho metálico. *Ex: crisântemo cobre.* |
| Laranja-queimado | Laranja acastanhado. *Ex: algumas variedades de helicônia.* |

### Vermelho

| Variação | Descrição |
|---|---|
| Vermelho-rosado | Vermelho claro com toque rosa. *Ex: cosmos, dianthus claro.* |
| Vermelho-cereja | Vermelho vivo e saturado. *Ex: tulipa vermelha.* |
| Vermelho-escarlate | Vermelho puro e intenso. *Ex: papoula, antúrio.* |
| Vermelho-carmim | Vermelho profundo com toque azulado. *Ex: rosa vermelha.* |
| Vermelho-bordeaux | Vermelho muito escuro, quase vinho. *Ex: dália escura, peônia.* |
| Vermelho-enegrecido | Tão escuro que parece quase preto. *Ex: rosa black magic.* |

### Rosa

| Variação | Descrição |
|---|---|
| Rosa-bebê | Rosa muito claro, delicado. *Ex: cerejeira ornamental.* |
| Rosa-quartzo | Rosa claro com tom acinzentado. *Ex: peônia pálida.* |
| Rosa-chiclete | Rosa médio, vivo. *Ex: cravina rosa, zinnia.* |
| Rosa-coral | Rosa com toque alaranjado. *Ex: rosa coral, boca-de-leão coral.* |
| Rosa-pink | Rosa intenso, quase fúcsia. *Ex: gérbera pink, astromelia.* |
| Rosa-fúcsia | Rosa muito saturado e vibrante. *Ex: buganvília, cravina fúcsia.* |
| Rosa-magenta | Rosa com forte toque roxo. *Ex: orquídea dendrobium.* |

### Violeta / Roxo

| Variação | Descrição |
|---|---|
| Lilás | Roxo muito claro e suave. *Ex: lilás (Syringa), wisteria claro.* |
| Lavanda | Azul-roxo claro e acinzentado. *Ex: lavanda (Lavandula).* |
| Violeta-claro | Roxo claro sem toque azulado. *Ex: violeta africana clara.* |
| Violeta-médio | Roxo equilibrado. *Ex: íris roxa, crisântemo lilás.* |
| Violeta-intenso | Roxo saturado e vivo. *Ex: allium, verbena.* |
| Roxo-escuro | Roxo profundo. *Ex: íris escura, tulipa roxa.* |
| Roxo-enegrecido | Roxo quase preto. *Ex: peônia chocolate, tulipa queen of night.* |

### Azul

> **Nota:** azul puro é raro em flores — a maioria tem toque roxo ou é tingida artificialmente.

| Variação | Descrição |
|---|---|
| Azul-celeste | Azul muito claro. *Ex: miosótis (forget-me-not).* |
| Azul-periwinkle | Azul com toque lilás. *Ex: pervinca, agapanto claro.* |
| Azul-médio | Azul moderado. *Ex: hortênsia azul, centaurea.* |
| Azul-cobalto | Azul intenso. *Ex: delfínio azul.* |
| Azul-índigo | Azul muito escuro com toque violeta. *Ex: aconite.* |

### Verde

> **Nota:** pétalas verdes frequentemente indicam mutação ou espécie ornamental.

| Variação | Descrição |
|---|---|
| Verde-claro | Quase amarelo-esverdeado. *Ex: gladíolo verde-limão.* |
| Verde-menta | Verde suave e fresco. *Ex: helébore verde.* |
| Verde-médio | Verde típico de folha. *Ex: zinnia verde-envy.* |
| Verde-escuro | Verde profundo. *Ex: algumas orquídeas exóticas.* |

### Branco

| Variação | Descrição |
|---|---|
| Branco-puro | Sem pigmentação aparente. *Ex: margarida branca, lírio branco.* |
| Branco-creme | Branco com leve toque amarelado. *Ex: rosa branca clássica.* |
| Branco-esverdeado | Branco com subtom verde. *Ex: helébore branco, antúrio branco.* |
| Branco-rosado | Branco com leve rubor. *Ex: cerejeira ornamental clara.* |

### Multicolorida / Bicolor

| Variação | Descrição |
|---|---|
| Bicolor contrastante | Duas cores bem distintas em regiões separadas. *Ex: tulipa bicolor.* |
| Degradê (ombré) | Transição gradual de uma cor para outra. *Ex: rosa ombré.* |
| Manchada / Pintalgada | Cor de base com manchas ou pintas de outra cor. *Ex: orquídea tigre.* |
| Listrada | Faixas paralelas de cores alternadas. *Ex: tulipa listrada.* |
| Com olho central | Cor diferente no centro da flor. *Ex: anêmona, prímula.* |

## 1.2 Formato e Estrutura das Pétalas

### Forma individual da pétala

| Forma | Descrição |
|---|---|
| Arredondada | Pétala com contorno circular ou oval suave. |
| Espatulada | Estreita na base e larga e arredondada no ápice. |
| Lanceolada | Longa e estreita, pontiaguda na ponta. |
| Linear | Muito estreita e comprida, de largura uniforme. |
| Fimbriada / Franjada | Bordas recortadas ou com franjas. *Ex: tulipa fimbriada.* |
| Ondulada | Bordas em forma de onda. *Ex: papoula, orquídea.* |
| Encurvada | Pétala que se curva para dentro no ápice. |
| Reflexa | Pétala que se dobra para trás, em direção ao pedúnculo. |

### Textura da pétala

| Textura | Descrição |
|---|---|
| Lisa | Superfície uniforme, sem relevo. |
| Velutinosa / Aveludada | Microestruturas que dão aparência de veludo. *Ex: rosa.* |
| Cerosa | Aspecto brilhante e levemente gorduroso. |
| Translúcida | Deixa passar luz, aparência delicada. *Ex: papoula, begônia.* |
| Papilosa | Superfície com pequenas protuberâncias. |

### Número de pétalas

| Quantidade | Descrição |
|---|---|
| 1 pétala | Extremamente raro. |
| 3 pétalas | Típico de monocotiledôneas (trímeras). *Ex: trillium.* |
| 4 pétalas | Tetrâmeras. *Ex: mostarda, papoula.* |
| 5 pétalas | Pentâmeras — o mais comum. *Ex: rosa silvestre, cerejeira.* |
| 6 pétalas | Típico de liláceas/monocots. *Ex: lírio, tulipa.* |
| 8 a 12 pétalas | *Ex: cosmo, anemona.* |
| 13 a 21 pétalas | *Ex: margarida simples, girassol ornamental.* |
| Numerosas (> 21) | *Ex: peônia dobrada, rosa dobrada, dália decorativa.* |

### Disposição (corola)

| Disposição | Descrição |
|---|---|
| Simples | Uma única camada de pétalas. |
| Semi-dupla | Duas a três camadas de pétalas. |
| Dupla / Dobrada | Múltiplas camadas; estames frequentemente transformados em pétalas. |
| Pompom | Pétalas curtas e densas formando uma esfera. *Ex: dália pompom.* |

## 1.3 Formato Geral da Flor (Corola)

| Formato | Descrição |
|---|---|
| Estrelada | Pétalas abertas formando uma estrela. *Ex: jasmim, borragem.* |
| Tubular | Pétalas fundidas formando um tubo. *Ex: lobélia, fucsia.* |
| Campanulada (em sino) | Pétalas formam uma campânula. *Ex: sino-de-natal, lisiantus.* |
| Infundibuliforme | Formato de funil, alargando da base para a abertura. *Ex: petúnia.* |
| Rotácea | Pétalas abertas e planas, corola em roda. *Ex: prímula.* |
| Bilabial (dois lábios) | Corola dividida em lábio superior e inferior. *Ex: snapdragon.* |
| Papilionácea | Formato de borboleta — típico de leguminosas. *Ex: ervilha, lupinus.* |
| Em espata | Uma pétala modificada envolve a espádice. *Ex: antúrio, copo-de-leite.* |
| Ligulada | Pétalas fundidas em uma fita. *Ex: flores da margem do girassol.* |
| Capitular (cabeça) | Inflorescência densa que parece uma única flor. *Ex: girassol, margarida, dália.* |

---

## 1.4 Verticilos Florais (Completude)

| Tipo | Descrição |
|---|---|
| Completa | Possui os quatro verticilos: cálice (sépalas), corola (pétalas), androceu (estames) e gineceu (pistilo). |
| Incompleta | Falta ao menos um verticilo. |
| — Aclamídea / Nua | Sem perianto (sem cálice e sem corola). *Ex: salgueiro.* |
| — Monoclamídea | Possui apenas cálice ou apenas corola. *Ex: beterraba.* |
| — Diclamídea | Possui ambos cálice e corola (a maioria das flores ornamentais). |

---

## 1.5 Simetria Floral

| Simetria | Descrição |
|---|---|
| Actinomorfa (radial) | Dois ou mais planos de simetria; flor "em estrela". *Ex: rosa, margarida, tulipa.* |
| Zigomorfa (bilateral) | Apenas um plano de simetria. *Ex: orquídea, ervilha, boca-de-leão, violeta.* |
| Assimétrica | Nenhum plano de simetria. Extremamente raro. *Ex: Canna.* |

---

## 1.6 Sexualidade Floral

| Tipo | Descrição |
|---|---|
| Perfeita (hermafrodita) | Possui androceu e gineceu na mesma flor. *Ex: rosa, tulipa, girassol.* |
| Imperfeita (unissexual) | Possui apenas um dos verticilos reprodutivos. |
| — Pistilada (feminina) | Apenas gineceu. *Ex: flores femininas do mamão.* |
| — Estaminada (masculina) | Apenas androceu. *Ex: flores masculinas do milho.* |

### Plantas quanto ao sexo

| Tipo | Descrição |
|---|---|
| Monoica | Flores masculinas e femininas no mesmo indivíduo. *Ex: mamona, milho, abóbora.* |
| Dioica | Flores masculinas e femininas em indivíduos separados. *Ex: mamoeiro, kiwi, aspargo.* |
| Andrógina | Apenas flores hermafroditas. *Ex: rosa, tulipa, girassol.* |
| Poligâmica | Possui flores hermafroditas e unissexuais. *Ex: algumas espécies de freixo e bordo.* |

---

## 1.7 Número de Peças Florais (Merismo)

| Tipo | Descrição |
|---|---|
| Trímera | Peças florais em múltiplos de 3. Típico de monocotiledôneas. *Ex: lírio (6 tépalas), tulipa (6 tépalas), íris.* |
| Tetrâmera | Peças florais em múltiplos de 4. *Ex: mostarda (4 pétalas), losna.* |
| Pentâmera | Peças florais em múltiplos de 5. Típico de eudicotiledôneas. *Ex: rosa (5 sépalas + 5+ pétalas), papoula.* |

---

## 1.8 Inflorescência (Agrupamento de Flores)

| Tipo | Descrição |
|---|---|
| Flor solitária | Uma única flor por pedúnculo. *Ex: papoula, tulipa.* |
| Racemo | Flores com pedicelo próprio ao longo de um eixo central. *Ex: gladíolo.* |
| Espiga | Como o racemo, mas flores sésseis (sem pedicelo). *Ex: trigo ornamental.* |
| Umbela | Todos os pedicelos partem do mesmo ponto. *Ex: cenoura ornamental, alho.* |
| Corimbo | Flores em alturas diferentes mas ápices ao mesmo nível. *Ex: inflorescência da maçã.* |
| Cimeira | Eixo principal termina em flor; ramificações laterais continuam. *Ex: borragem.* |
| Capítulo | Flores sésseis sobre receptáculo alargado; parece uma única flor. *Ex: girassol, margarida, dália.* |
| Cacho (panícula) | Racemo ramificado. *Ex: lilás, glicínia.* |
| Espata + Espádice | Inflorescência em espádice envolta por uma espata colorida. *Ex: antúrio, copo-de-leite.* |

---

## 1.9 Aroma

### Intensidade

| Nível | Descrição |
|---|---|
| Inodora | Sem aroma perceptível. *Ex: tulipa, margarida.* |
| Suave | Aroma leve, perceptível apenas de perto. |
| Moderado | Aroma presente sem ser dominante. |
| Intenso | Aroma forte, perceptível a distância. |
| Muito intenso | Aroma avassalador. *Ex: gardênia, jasmim.* |

### Categoria do aroma

| Categoria | Descrição |
|---|---|
| Floral-doce | Aroma adocicado, clássico de perfumaria. *Ex: rosa, gardênia.* |
| Cítrico | Toque de limão ou laranja. *Ex: magnólia, algumas orquídeas.* |
| Especiado | Aroma quente, tipo cravo ou canela. *Ex: cravina, dianthus.* |
| Frutal | Reminiscente de fruta. *Ex: mimosa, algumas azaleias.* |
| Mel / Néctar | Doce e viscoso. *Ex: lavanda, trevo.* |
| Fresco / Verde | Herbal, gramíneo. *Ex: algumas verbenas.* |
| Almiscarado | Aroma profundo e animal. *Ex: lírio-do-vale.* |
| Desagradável | Atrai polinizadores como moscas. *Ex: Rafflesia, Amorphophallus.* |

### Período de liberação do aroma

| Período | Descrição |
|---|---|
| Diurno | Aroma liberado durante o dia. |
| Noturno | Aroma liberado à noite. *Ex: dama-da-noite, jasmim-noturno.* |
| Constante | Aroma presente continuamente. |

---

## 1.10 Polinização

### Por insetos (entomofilia)

| Polinizador | Características da flor |
|---|---|
| Abelhas | Flores azuis, roxas, amarelas; néctar acessível; aroma doce. |
| Borboletas | Flores com plataforma de pouso; cores vivas; néctar profundo. |
| Mariposas | Flores brancas ou pálidas; aroma noturno intenso. |
| Besouros | Flores grandes, abertas, aroma forte (por vezes fétido). |
| Moscas | Flores com aroma de carne ou apodrecimento. |

### Por vertebrados

| Polinizador | Características da flor |
|---|---|
| Pássaros (ornitofilia) | Flores tubulares, vermelhas ou laranjas, sem aroma. *Ex: helicônia, hibisco, fúcsia.* |
| Morcegos (quiropterofilia) | Flores noturnas, grandes, brancas ou cremosas, aroma forte. *Ex: baobá, algumas cactáceas.* |

### Outros

| Tipo | Descrição |
|---|---|
| Pelo vento (anemofilia) | Flores pequenas, sem pétalas vistosas, pólen abundante e leve. *Ex: grama, carvalho, bétula.* |
| Por água (hidrofilia) | Raro; plantas aquáticas. *Ex: eelgrass.* |

# 2. Folhas

## 2.1 Estrutura do Limbo

| Tipo | Descrição |
|---|---|
| Simples | Lâmina foliar única, não dividida em folíolos. Pode ser inteira, lobada ou recortada na margem. |
| Composta | Limbo dividido em folíolos distintos, cada um com aspecto de folha. |
| — Pinada | Folíolos dispostos aos pares ao longo de um eixo central (ráquis). |
| — — Imparipinada | Número ímpar de folíolos (termina com um folíolo apical). *Ex: rosa.* |
| — — Paripinada | Número par de folíolos (sem folíolo apical). *Ex: ervilha.* |
| — Bipinada | Cada folíolo é, ele mesmo, pinado. *Ex: sensitive (dormideira).* |
| — Palmada | Folíolos partem de um ponto central como dedos de uma mão. *Ex: mandioca, cannabis ornamental.* |
| — Trifoliada | Exatamente 3 folíolos. *Ex: trevo, feijão.* |

---

## 2.2 Formato do Limbo

| Formato | Descrição |
|---|---|
| Circular / Orbicular | Forma aproximadamente circular. *Ex: nastúrcio.* |
| Oval / Ovada | Forma de ovo, parte mais larga próxima ao pecíolo. |
| Obovada | Forma de ovo invertido; parte mais larga afastada do pecíolo. |
| Elíptica | Forma de elipse, simétrica. |
| Lanceolada | Longa e estreita, com ponta aguda. *Ex: salgueiro, liriope.* |
| Oblanceolada | Lanceolada invertida; mais larga no ápice. |
| Linear | Muito estreita e comprida, margens paralelas. *Ex: grama, íris.* |
| Acicular | Em forma de agulha. *Ex: pinheiro.* |
| Espatulada | Estreita na base e alargada e arredondada no ápice. |
| Reniforme | Em forma de rim. *Ex: violeta africana.* |
| Cordada | Base em formato de coração. *Ex: lilás, hortênsia.* |
| Sagitada | Base com dois lóbulos apontando para baixo (forma de seta). *Ex: sagitária.* |
| Hastada | Similar à sagitada, mas lóbulos apontam para os lados. |
| Lobada | Limbo dividido em lóbulos (recortes que não atingem a nervura). |
| — Trilobada | 3 lóbulos. *Ex: hera ornamental.* |
| — Palmatilobada | Lóbulos como dedos de uma mão. *Ex: mamoeiro.* |
| — Penatilobada | Lóbulos em pares ao longo de nervura central. *Ex: carvalho.* |
| Laciniada | Recortada em segmentos estreitos e irregulares. *Ex: crisântemo.* |

---

## 2.3 Margem do Limbo

| Margem | Descrição |
|---|---|
| Inteira | Margem lisa, sem recortes. |
| Dentada | Margem com dentes agudos perpendiculares. *Ex: rosa.* |
| Serrada | Dentes inclinados para o ápice (como uma serra). |
| Crenada | Margem com ondulações arredondadas (festonada). |
| Ondulada | Margem com ondas suaves no plano vertical. |
| Ciliada | Margem com pelos finos (cílios). *Ex: violeta.* |
| Espinhosa | Margem com espinhos rígidos. *Ex: azevinho.* |

---

## 2.4 Nervação

| Tipo | Descrição |
|---|---|
| Paralelinérvea | Nervuras paralelas entre si; típica de monocotiledôneas. *Ex: grama, milho, lírio, tulipa.* |
| Peninérvea (reticulada) | Nervura central com ramificações laterais formando rede. Típica de eudicotiledôneas. *Ex: rosa, hortênsia, magnólia.* |
| — Simples | Uma única nervura central proeminente. |
| — Arco-nervada | Nervuras secundárias partem da base e convergem no ápice. *Ex: hosta, helicônia.* |
| Palmatinérvea | Várias nervuras principais partem da base como dedos da mão. *Ex: mamoeiro, mandioca.* |
| Uninérvea | Apenas uma nervura central (folhas muito simples). *Ex: musgos.* |

---

## 2.5 Cor da Folha

| Cor | Exemplos |
|---|---|
| Verde-claro | *Ex: hostas de variedade clara, helébore.* |
| Verde-médio | A maioria das folhas ornamentais. |
| Verde-escuro | *Ex: gardênia, camélia, magnólia.* |
| Verde-azulado | *Ex: hosta blue, eucalipto.* |
| Verde-amarelado | *Ex: hostas chartreuse, áurea.* |
| Variegada branca | Manchas ou bordas brancas sobre verde. *Ex: hosta francee.* |
| Variegada amarela | Manchas ou bordas amarelas. *Ex: hosta gold standard.* |
| Variegada prateada | Tons prateados. *Ex: lamium, pulmonaria.* |
| Avermelhada / Roxa | *Ex: bananeira-roxa, begônia rex, planta-preta.* |
| Bronzeada | *Ex: sedge bronze, alguns crisântemos.* |
| Cinza / Prateada | *Ex: artemísia, senécio, echeveria.* |

---

## 2.6 Textura e Superfície da Folha

| Textura | Descrição |
|---|---|
| Lisa e brilhante | Superfície sem pelos, com reflexo. *Ex: camélia, magnólia.* |
| Lisa e opaca | Sem pelos e sem brilho. *Ex: hosta, violeta africana.* |
| Pubescente | Coberta de pelos curtos e macios. *Ex: verbena, salva.* |
| Tomentosa | Pelos densos formando uma camada como feltro. *Ex: orelha-de-coelho.* |
| Rugosa | Superfície com relevos irregulares. *Ex: salva, bálsamo.* |
| Bolhosa | Superfície com protuberâncias arredondadas (bolhas). *Ex: couve-salva.* |
| Cerosa | Revestimento de cera (pruinosa). *Ex: echeveria, sedum.* |
| Viscosa / Pegajosa | Secreção aderente. *Ex: verbascum, algumas primulas.* |

---

## 2.7 Folhas Modificadas (Especializadas)

| Tipo | Descrição |
|---|---|
| Bráctea | Folha colorida que envolve ou sustenta a inflorescência; frequentemente confundida com pétala. *Ex: buganvília (roxa/rosa), antúrio (vermelho), poinsétia (vermelha), helicônia (laranja).* |
| Espata | Bráctea grande que envolve a espádice. *Ex: copo-de-leite, antúrio.* |
| Espinhos foliares | Folhas ou estípulas transformadas em espinhos de proteção. *Ex: cactos (aréolas), berberis.* |
| Gavinhas foliares | Folha ou parte dela transformada em gavinha de fixação. *Ex: ervilha (folíolos apicais), Lathyrus.* |
| Pitchers (urnas) | Folha transformada em armadilha com líquido digestivo. *Ex: Nepenthes, Sarracenia.* |
| Suculentas | Folhas espessadas para armazenamento de água. *Ex: Sedum, Echeveria, Agave.* |
| Escamas | Folhas reduzidas e não fotossintéticas (em bulbos). *Ex: tulipa, lírio.* |
| Sem modificação | Folhas fotossintéticas convencionais. |

# 3. Caules

## 3.1 Consistência / Tipo Básico

| Tipo | Descrição |
|---|---|
| Herbáceo | Caule mole, verde, sem lignificação significativa. *Ex: margarida, crisântemo, gerânio.* |
| Sublenhoso | Base levemente lenhosa, ramos jovens herbáceos. *Ex: lavanda, pelargônio.* |
| Lenhoso (arbusto) | Caule lenhoso mas ramificado desde a base, porte baixo a médio. *Ex: rosa arbusto, azaleia, hortênsia.* |
| Lenhoso (árvore) | Tronco único definido, porte maior. *Ex: cerejeira ornamental, magnólia-árvore, jacarandá.* |
| Suculento | Caule espessado para armazenamento de água. *Ex: cactos ornamentais, alguns crásulas.* |
| Trepador | Caule que se apoia em suportes (por gavinhas, volúvel ou raízes aéreas). *Ex: buganvília, wisteria, jasmin-trepadeira.* |
| Rasteiro / Prostrado | Caule que cresce horizontalmente rente ao solo. *Ex: verbena rasteira, algumas rosas groundcover.* |
| Rizomatoso | Caule subterrâneo horizontal que emite brotos. *Ex: íris, lírio-do-vale, canná.* |
| Bulboso | Caule muito reduzido na base de um bulbo. *Ex: tulipa, narciso, allium.* |
| Tuberoso | Caule subterrâneo espessado (tubérculo). *Ex: dália, begônia tuberosa.* |

---

## 3.2 Superfície do Caule

| Superfície | Descrição |
|---|---|
| Liso | Sem pelos ou protuberâncias visíveis. |
| Piloso / Pubescente | Coberto de pelos finos. |
| Aculeado | Com acúleos (estruturas que se desprendem facilmente). *Ex: rosa.* |
| Espinhoso | Com espinhos verdadeiros (prolongamentos do caule). *Ex: bougainvillea.* |
| Estriado | Com sulcos longitudinais. *Ex: salgueiro, algumas herbáceas.* |
| Alado | Com expansões laminares (alas) ao longo do caule. *Ex: lathyrus.* |
| Ceroso / Pruinoso | Revestimento de cera esbranquiçada. *Ex: algumas euphorbiáceas.* |
| Nodoso | Com nós muito evidentes (internós curtos). *Ex: bambu ornamental.* |

---

## 3.3 Seção Transversal do Caule

| Seção | Descrição |
|---|---|
| Cilíndrico | Seção circular — o mais comum. |
| Quadrangular | Seção quadrada. *Ex: lamiáceas: menta, salva, lavanda.* |
| Triangular | Seção triangular. *Ex: algumas ciperáceas ornamentais.* |
| Achatado (fistuloso) | Caule achatado ou com costelas. *Ex: algumas orquídeas.* |
| Oco | Interior vazio. *Ex: umbelíferas, bambusóideas.* |

---

## 3.4 Cor do Caule / Ramo

| Cor | Exemplos |
|---|---|
| Verde | Caule fotossintético. *Ex: herbáceas em geral.* |
| Verde-avermelhado | *Ex: peônia (jovem), algumas rosas.* |
| Marrom-claro | Caule lenhoso jovem. *Ex: hortênsia.* |
| Marrom-escuro | Caule lenhoso maduro. *Ex: rosa antiga, cerejeira.* |
| Cinza | *Ex: salgueiro-chorão, algumas árvores ornamentais.* |
| Vermelho / Vinho | *Ex: cornus ornamental de caule vermelho.* |
| Preto / Enegrecido | *Ex: bambu-preto ornamental.* |
| Branco / Prateado | *Ex: betula ornamental (bétula), álamo.* |

---

## 3.5 Caules Modificados (Estruturas Especializadas)

| Tipo | Descrição |
|---|---|
| Rizoma | Caule horizontal subterrâneo com reservas e gemas. *Ex: íris, canná.* |
| Bulbo túnica (coberto) | Bulbo com camada externa seca (túnica). *Ex: tulipa, narciso.* |
| Bulbo escamoso | Sem túnica; folhas escamosas expostas. *Ex: lírio.* |
| Cormo | Caule subterrâneo maciço e sólido, sem escamas. *Ex: gladíolo, crocus, freesia.* |
| Tubérculo | Caule subterrâneo espessado sem escamas nem túnica. *Ex: dália, begônia tuberosa, anemona.* |
| Estolão | Caule rasteiro aéreo que emite raízes e brotos nos nós. *Ex: morangueiro ornamental, violeta rasteira.* |
| Gavinha caulinar | Caule modificado para fixação em suporte. *Ex: videira ornamental, maracujá.* |
| Cladódio | Caule achatado e fotossintético que imita folha. *Ex: cactus de natal (Schlumbergera), pitaia.* |
| Sem modificação | Caule típico, sem especialização estrutural. |

# 4. Raízes

## 4.1 Tipo de Sistema Radicular

| Tipo | Descrição |
|---|---|
| Pivotante | Raiz principal central e dominante com raízes laterais menores. Típica de eudicotiledôneas. *Ex: rosa, girassol.* |
| Fasciculado | Conjunto de raízes de tamanho similar, sem raiz principal. Típico de monocotiledôneas. *Ex: tulipa, lírio, grama.* |
| Misto | Sistema intermediário. *Ex: muitas herbáceas cultivadas.* |

---

## 4.2 Raízes Modificadas

| Tipo | Descrição |
|---|---|
| Tuberosa | Raiz espessada com reserva nutritiva. *Ex: dália (raízes tuberosas em grupo), batata-doce ornamental.* |
| Aérea | Raiz que cresce fora do solo (sustentação ou absorção de umidade). *Ex: orquídeas epífitas, hera em parede.* |
| Escora | Raízes laterais que saem do caule e funcionam como suporte. *Ex: milho ornamental, algumas palmeiras.* |
| Haustorial | Raízes parasitas que penetram no hospedeiro. *Ex: cipó-chumbo.* |
| Sem modificação | Sistema radicular típico. |

# 5. Características Gerais da Planta

## 5.1 Porte / Altura Típica

| Porte | Altura | Exemplos |
|---|---|---|
| Miniatura | Até 15 cm | *Ex: violeta africana mini, crocus.* |
| Rasteira / Baixa | 15 a 40 cm | *Ex: verbena, lobulária, impatiens.* |
| Média | 40 cm a 1 m | *Ex: gérbera, crisântemo, lavanda.* |
| Alta | 1 a 2 m | *Ex: girassol, delfínio, gladíolo.* |
| Arbustiva | 1 a 3 m | *Ex: rosa arbusto, hortênsia, azaleia.* |
| Arbórea | Acima de 3 m | *Ex: cerejeira, magnólia-árvore, jacarandá.* |

---

## 5.2 Ciclo de Vida

| Ciclo | Descrição |
|---|---|
| Anual | Completa o ciclo em um ano (germinação → floração → semente → morte). *Ex: girassol, zinnia, cosmos, petúnia.* |
| Bienal | Primeiro ano: crescimento vegetativo. Segundo: floresce e morre. *Ex: foxglove (digital), verbascum, cardo.* |
| Perene sempre-verde | Vive por muitos anos; mantém folhagem o ano todo. *Ex: gardênia, camélia.* |
| Perene caduca | Vive por muitos anos; perde folhas no outono/inverno. *Ex: hosta, peônia, íris.* |
| Geófita | Perene com órgão de reserva subterrâneo; dorme no período desfavorável. *Ex: tulipa, narciso, dália.* |

---

## 5.3 Época de Floração

| Época | Exemplos |
|---|---|
| Primavera | *Ex: tulipa, narciso, cerejeira, wisteria.* |
| Verão | *Ex: girassol, rosa (pico), gérbera, lavanda.* |
| Outono | *Ex: crisântemo, dália, anemona japonesa.* |
| Inverno | *Ex: camélia, helébore, amor-perfeito inverno.* |
| Floração longa | Primavera ao outono. *Ex: petúnia, impatiens, zinnia.* |
| Reflorescente | Floresce em múltiplos períodos no ano. *Ex: rosa reflorescente.* |
| Noturna | Flores que abrem à noite. *Ex: dama-da-noite, cactus-da-lua.* |

---

## 5.4 Habitat / Origem

| Habitat | Descrição |
|---|---|
| Temperado | Regiões de clima temperado com estações definidas. *Ex: tulipa, peônia.* |
| Mediterrâneo | Verões secos e quentes, invernos amenos e úmidos. *Ex: lavanda, cistus.* |
| Tropical | Clima quente e úmido o ano todo. *Ex: helicônia, antúrio, strelitzia.* |
| Subtropical | Quente com alguma sazonalidade. *Ex: hibisco, buganvília.* |
| Alpino / Montanha | Regiões frias e altitude elevada. *Ex: edelweiss, gentiana.* |
| Aquático / Paludoso | Beira d'água ou ambientes alagados. *Ex: lírio-d'água, íris aquática.* |
| Desértico / Árido | Adaptado à seca. *Ex: cactos ornamentais, agave.* |
| Epífita | Cresce sobre outras plantas sem parasitá-las. *Ex: orquídeas, bromélia.* |

---

## 5.5 Toxicidade

| Nível | Descrição |
|---|---|
| Não tóxica | Segura para humanos e animais domésticos. |
| Tóxica para animais | Perigosa para cães e/ou gatos se ingerida. *Ex: tulipa (bulbo), lírio (gatos), foxglove.* |
| Tóxica para humanos | Ingestão pode causar intoxicação. *Ex: delfínio, aconite, digital, oleandro.* |
| Suco/látex irritante | Contato com seiva causa irritação de pele ou mucosas. *Ex: eufórbia, hera, primula (algumas).* |
| Altamente venenosa | Todas as partes são tóxicas, inclusive por contato prolongado. *Ex: aconite (acônito), cicuta ornamental.* |
